# Users Dashboard

## Основные возможности

### Аутентификация

- Вход в систему через DummyJSON API
- Хранение JWT токенов в localStorage
- Защита маршрутов (дашборд доступен только авторизованным пользователям)
- Выход из системы с очисткой токенов

### Поиск и фильтрация

- Поиск пользователей по имени, email или username
- Фильтрация по отделам компании
- Пагинация с умной навигацией (10 пользователей на страницу)

### CRUD операции

- Добавление нового пользователя через форму с валидацией
- Редактирование существующего пользователя
- Удаление пользователя с подтверждением
- Уведомления о результатах операций (toast)

## API интеграция

| Эндпоинт          | Метод  | Описание                        |
| ----------------- | ------ | ------------------------------- |
| /auth/login       | POST   | Аутентификация пользователя     |
| /auth/me          | GET    | Получение текущего пользователя |
| /users            | GET    | Получение списка пользователей  |
| /users/search     | GET    | Поиск пользователей             |
| /users/filter     | GET    | Фильтрация пользователей        |
| /users/add        | POST   | Добавление пользователя         |
| /users/{id}       | PUT    | Обновление пользователя         |
| /users/{id}       | DELETE | Удаление пользователя           |
| /users/{id}/carts | GET    | Корзины пользователя            |
| /users/{id}/posts | GET    | Посты пользователя              |
| /users/{id}/todos | GET    | Задачи пользователя             |

## UI/UX особенности

- Адаптивный дизайн

- Toast уведомления

- Диалог подтверждения перед удалением
