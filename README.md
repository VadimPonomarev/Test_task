Приложение использует DummyJSON API для работы с данными:

Эндпоинт Метод Описание
/auth/login POST Аутентификация пользователя
/auth/me GET Получение текущего пользователя
/users GET Получение списка пользователей
/users/search GET Поиск пользователей
/users/filter GET Фильтрация пользователей
/users/add POST Добавление пользователя
/users/{id} PUT Обновление пользователя
/users/{id} DELETE Удаление пользователя
/users/{id}/carts GET Корзины пользователя
/users/{id}/posts GET Посты пользователя
/users/{id}/todos GET Задачи пользователя
